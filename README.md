# Typing Speed Test

In this project, we will build a responsive typing speed test application using HTML, CSS, and JavaScript. The application will display a random paragraph for users to type, track their input in real time, and calculate typing speed and accuracy. You will also add controls to restart or stop the test anytime.

The project contains the following three main files:

<ul>
  <li>/usercode/templates/index.html: Defines the application’s structure, including headings, input area, buttons, and results section.</li>
  <li>/usercode/static/styles.css: Controls the visual layout and spacing, creating a clean and user-friendly design.</li>
  <li>/usercode/static/index.js: Adds interactivity by handling paragraph selection, input tracking, timer logic, and performance calculations.</li>
</ul>
